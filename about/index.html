About YAMLScript

YAMLScript is a programming language that combines the simplicity of YAML with the functional power of Clojure. It is a superset of YAML, meaning that any valid YAML document is a valid YAMLScript program.

NOTE: YAMLScript is a work in progress, and is not yet ready for production use.
